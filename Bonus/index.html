Sincronizzazione semplice dei file

Una Sincronizzazionerapida che ti permette di risparmiare spazio

Sincronizza i file su tutti i dispositivi e le piattaforme

È facile rendere accessibili i tuoi file sul percorso casa-lavoro o quando sei in vacanza. Salva un file nella cartella Dropbox sul tuo computer e verrà sincronizzato automaticamente sui tuoi dispositivi mobili. La sincronizzazione dei file su cloud è disponibile su più dispositivi e piattaforme, da computer Windows e Mac a dispositivi mobili come iPhone, iPad e dispositivi Android, tramite l'app mobile di Dropbox.

I file salvati o aggiornati di recente vengono sincronizzati automaticamente su tutti i dispositivi, così non dovrai perdere tempo a inviare e-mail con le versioni più aggiornate ai collaboratori. E puoi avere la conferma che i tuoi file importanti siano completamente sincronizzati controllando il segno di spunta verde.

La sincronizzazione dei file è sempre sotto controllo

Hai aggiornato un file troppo presto? Non preoccuparti, non tutto è perduto. Le versioni precedenti dei tuoi file rimangono salvate per un massimo di 30 giorni, o addirittura 180 per gli utenti Dropbox Business, così potrai sempre recuperare una versione precedente della sincronizzazione dei tuoi file.

Ottieni l'accesso online su tutti i tuoi dispositivi

Con l'app desktop, i file e le cartelle sincronizzati in locale sono disponibili anche quando non hai accesso alla connessione internet. Quando tornerai online, Dropbox sincronizzerà automaticamente le tue cartelle e i tuoi file con tutte le modifiche più recenti. Puoi anche selezionare i file a cui desideri accedere offline sullo smartphone Android, su iPhone e su iPad.

Risparmia spazio

Con la sincronizzazione intelligente di Dropbox, puoi liberare spazio utile sull'hard disk caricando i file nel cloud. Anche se sincronizzati e archiviati nel cloud , tutti i tuoi file e cartelle saranno ancora visibili dal tuo computer, anche se non occuperanno praticamente spazio. Quando vorrai accedervi, i file si sincronizzeranno automaticamente con il tuo hard disk, ma solo quando ne hai bisogno.

Domande frequenti

Che cos'è la sincronizzazione?

Per "sincronizzazione" si intende un evento che si verifica in più di un luogo contemporaneamente. In termini di tecnologia, quando si sincronizza un dispositivo, ad esempio un telefono o un tablet, con il computer, tutti i dati del computer vengono automaticamente resi accessibili su tale dispositivo. In questo modo, puoi rilassarti sapendo che tutti i tuoi dati, dalle foto personali ai file di lavoro, sono disponibili su diversi dispositivi. La sincronizzazione ti aiuta a proteggerti ulteriormente dalla perdita di dati, in quanto i tuoi file vengono salvati in più di un posto. Con la sincronizzazione intelligente di Dropbox, puoi anche risparmiare spazio sul disco rigido rimuovendo vecchie cartelle che non usi regolarmente e archiviandole nel cloud.

Che cos'è la sincronizzazione intelligente di Dropbox?

La sincronizzazione intelligente di Dropbox è una funzione che consente di risparmiare spazio sul disco rigido rimuovendo i file e le cartelle non aggiornati dal disco rigido e archiviandoli nel cloud. Grazie alla sincronizzazione intelligente puoi visualizzare tutti gli elementi presenti nel tuo account Dropbox e accedervi direttamente dal desktop, senza occupare spazio sul disco rigido del computer.

Come si sincronizzano i file e le cartelle su Dropbox?

In Dropbox, anziché utilizzare la sincronizzazione intelligente, puoi scegliere di utilizzare la sincronizzazione selettiva. Questo tipo di sincronizzazione ti consente di liberare spazio sul disco rigido senza eliminare alcun elemento dal tuo account Dropbox. Per utilizzare la sincronizzazione selettiva è necessario attivare questa funzione dall'applicazione di Dropbox su computer Windows o Mac. Una volta attivata la sincronizzazione selettiva, potrai scegliere quali cartelle escludere dal computer; queste cartelle rimarranno comunque sul tuo account dropbox.com.

Come si utilizza la sincronizzazione selettiva su Dropbox?

Puoi sincronizzare i file e le cartelle presenti su Dropbox su tutti i dispositivi utilizzando l'applicazione di Dropbox. Accedi al tuo account Dropbox su ogni dispositivo, quindi aggiungi i file e le cartelle che desideri alla cartella Dropbox presente sul tuo computer, telefono o tablet. La versione più recente di tutti i tuoi file e cartelle sarà disponibile su tutti i dispositivi.
